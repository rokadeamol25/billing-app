Troubleshooting

  • Make sure PostgreSQL is installed and running
  • Check the database credentials in server/.env
  • To start the React app, you need to resolve the PowerShell execution policy issue
  • Try running PowerShell as administrator and execute: Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
